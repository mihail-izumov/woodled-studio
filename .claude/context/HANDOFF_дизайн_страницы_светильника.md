## TL;DR

Функционально страница светильника работает. Алгоритм отлажен, фото подбираются точно под выбор юзера. Теперь нужно довести **визуал** — Hero-блок, нижнюю галерею, общую композицию страницы по дизайну.

В этом чате — **только UI**. Алгоритм и данные не трогать, иначе можно сломать тонкие правила.

## Что точно НЕ трогать (или трогать с пониманием)

- **`pickFxPhotos`** и набор HERO_FIELDS/GALLERY_FIELDS — отлажено через теггер, прогон 51 каталожной комбинации = 43 strict / 0 partial / 8 substitute / 0 empty.
- **`fxToConfig` и normalizeMount** — маппинг между UI-выборами и тегами фото. Случайная правка ломает сразу несколько fixtures.
- **`byFixture` strict-фильтр и иерархия curated→seed** — правило «курация главная». Менять только если продакт явно подтвердит.
- **`FAMILIES` в `fx-gallery.ts`** — `spot_s ≠ spot_l` (намеренно отдельные). Не путать с `catalog.ts FAMILIES`.
- **`provisional`/`isDirty`/`doSave`/`requestClose` поток** в FxEditor — Hero только читает `build`, не пишет. UI правки должны соблюдать это (никаких `@input="upBuild(...)"` от Hero-блока).
- **`id="fx-interiors"`** на обёртке нижней галереи — якорь CTA. Не убирать.

## Что можно трогать (UI-правки)

### `FxHeroGallery.vue`

- Высота квадрата-героя (сейчас `aspect-ratio: 1`).
- Бейджи слоёв: позиция, фон, шрифт, цвета. Текст — менять через `MATCH_BADGE` в `fx-gallery.ts`.
- Стрелки prev/next: размер, иконки, эффекты.
- Точки пагинации: позиция, размер, цвета.
- CTA-слайд: заголовок, размер мини-сетки, цвет пустых cells, кнопка.
- Lightbox: переехать на старый `gallery/Lightbox.vue` если хотим переиспользования (или оставить минимальный).
- Disclaimer: позиция, шрифт, иконки.

### `FxEditor.vue` (макет SUMMARY)

- Расстояния между Hero / Plate / Комплектация / Save / Share / Gallery.
- Толщина/радиусы карточек.
- Скроллпоинты, sticky-плашки.

### Старая `gallery/GallerySection.vue` + `PhotoCard.vue`

- Можно править если нужно, но это общий компонент (используется в App/RoomDetail). Лучше клонировать копию специально для FxEditor если правки крупные.
